# br_med_challenge

> Create a virtualenv

```bash
    python -m venv .venv
```

---

> Load a virtualenv

```bash
    source .venv/bin/activate
```

---

> Update pip

```bash
    pip install --upgrade pip
```

---

> Install poetry

```bash
    pip install poetry
```
---

> Start br_med_db database

```bash
    docker compose up -d
```
